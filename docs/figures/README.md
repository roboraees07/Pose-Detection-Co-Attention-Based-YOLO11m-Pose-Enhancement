# Architecture diagrams

Add your five figures here after creating the GitHub repository.
Use these **exact filenames** so the README displays them correctly.

| # | Your diagram | Filename |
|---|--------------|----------|
| 1 | PDCA-YOLO11-Pose complete pipeline | `full_pipeline.png` |
| 2 | ECA | `eca.png` |
| 3 | CBAM | `cbam.png` |
| 4 | C2PSA | `c2psa.png` |
| 5 | PDCA | `pdca.png` |

**Format:** PNG or JPG, 300 dpi recommended.

After saving all five files:

```bash
git add docs/figures/
git commit -m "Add architecture diagrams"
git push
```

Then uncomment the image block in the root `README.md`.
